Short description

This application was made as a final project for iOS Mobile Developmen at LaSalle College. It uses an API made by other students to get the conjugation of French verbs and display it to the user.

Key features

API using, List French verbs, Login/Signup, Database using, Data Validation.

Applied Concepts

Object Oriented Programming, API, Responsivity, Changing Screens, Component reusability, Search button, Tableview.
