BMI Calculator

Repository

Short description

The BMI (Body Mass Index) calculator is designed to help you keep on track your health. Whenever you need to know how to manage personal information about your own weight, it can be an useful tool to help you.

Key features

Responsive buttons, Slider button to choose your height, Round buttons to add or subtract from your age and/or weight, Result screen and personalized message based on your BMI.

Applied Concepts

Object Oriented Programming, Slider Buttons, Responsivity, Math Library, Changing Screens, Stateful and Stateless Widgets, Component reusability.
