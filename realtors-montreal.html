Realtors Montreal

Repository

Short description

Realtors Montreal is a work in progress for a web application I am developping as practice on Laravel PHP Framework. Here I'm aiming to improve my abilities on back-end developping for web using a different framework other than .NET, pure PHP or others.

Key features

CRUD and list all properties, tenant accounts, and appartments. Keep track on appartment status (available or not). Schedule appointments and respond tenants' messages. Report events to property ownlers. Create accounts, list all available appartments.

Applied Concepts

Object-Oriented Programming, Controls, Validation, Create/Read/Update/Delete/List All.
