Short description

Bill Bee is a work in progress between me and my partner Bernadette. We are developping an application to manage personal bills where you can track your expenses and verify your own payments.

Key features

Add bills, Pay bills, Login and Signup, Navigate between pages, Add information to cloud database, Access everywhere.

Applied Concepts

Object Oriented Programming, Responsivity, Clouding, Changing Screens, API, Component reusability.
