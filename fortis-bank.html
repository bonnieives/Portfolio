Fortis Bank

Repository
Partner: Bernadette Nicole Fernando

Short description

Fortis Bank is a software to simulate a bank system. The user can either be a manager, who is able to add/remove accounts and customers, print the transactions made by an account and get the list of all accounts/customers as well. The customer can withdraw, deposit and transfer values between accounts. All of its information is stored on database using SQL Server.

Key features

Manager: List all accounts and customers, list all transactions from accounts and customers, Create/remove an account, Add/remove a customer.
Customer: Withdraw, Get balance, Transfer and Deposit values.
All: Login validation with protected password.

Applied Concepts

Object-Oriented Programming, Interfaces, Classes, Inheritance, Database, Validation.
