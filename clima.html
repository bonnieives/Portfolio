Short description

This application uses live data from Open Weather website to provide the temperature and weather conditions to the user. It is really important to state that nowadays applications cannot have your device position unless you allow.

Key features

API using, Live data, Search city by name, Locate device, Provide weather conditions and temperature.

Applied Concepts

Object Oriented Programming, API, Responsivity, Math Library, Changing Screens, Stateful and Stateless Widgets, Component reusability, APIs, Search button.
