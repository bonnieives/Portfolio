Short description

This application uses live data from CoinAPI.io to get real time information about currency exchanges of some crypto currencies to some country currencies. When the user selects the data from a dropdown menu, it changes the currenci exchange from Bitcoin, Ethereum and LTC to the selected currency from the dropdown.

Key features

API using, Live data, Currency exchange, Dropdown menu.

Applied Concepts

Object Oriented Programming, API, Responsivity, Math Library, Stateful and Stateless Widgets, Component reusability.
